  # Vrinda Store Sales Analysis - Excel Dashboard 2022

Interactive Excel dashboard analyzing Vrinda Store's 2022 sales across Amazon, Flipkart, Myntra, Ajio, and Meesho. Built with PivotTables, PivotCharts, and Slicers to identify key customer segments.

### Dashboard Preview
![Vrinda Store Dashboard](dashboard%20preview.png)

### Key Insights
1. **Women drive 64% of sales** vs 36% for men
2. **Top states by revenue**: Maharashtra ₹29.9L, Karnataka ₹26.4L, Uttar Pradesh ₹21.0L  
3. **Adult age group** contributes 34.6% of total orders — highest segment
4. **Top channels**: Amazon 35.48%, Myntra 23.36%, Flipkart 21.59%
5. **92% orders delivered** — only 2.8% cancellations

### Business Recommendation
Target women customers aged 30-49 in Maharashtra, Karnataka, and Uttar Pradesh with ads/offers on Amazon, Myntra, and Flipkart to maximize ROI.
